# 🌆 AlasQuatro: 100 Days in Metro Manila – CMS Website

## 📌 Overview
This repository contains the **Content Management System (CMS) Website** for *100 Days in Metro Manila*, a 3D single-player mobile role-playing game focused on environmental awareness.

The website serves as the **official promotional and information platform**, providing users with access to game details, downloadable content, and educational materials about pollution and environmental responsibility.

---

## 🎮 About the Game
**100 Days in Metro Manila** is a narrative-driven action-adventure RPG that challenges players to restore a post-polluted version of Metro Manila within a limited timeframe.

The game integrates:
- 🌫️ Air Pollution (Smog District)
- 🌊 Water Pollution (Toxic Waterway)
- 🗑️ Land Pollution (Waste Grounds)

Players experience the consequences of environmental neglect through gameplay, encouraging awareness and responsible decision-making.

## 🔐 Admin Panel
- Accessible via **private link only**
- Full control over:
  - Website content updates  
  - Media uploads (images, assets)  
  - Game file updates/download link  
  - Website structure modifications

## 👤 User Access
- Visitors can browse public pages without login
